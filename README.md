# Tic Tac Toe 

## Features
   * Game boards 3x3 4x4 5x5

## Start server
   **To start server use** ``stack run server <port>``
   
## Start client 
   **To start client use** ``stack run client <host> <port>`` 
   
   **port** - server port name
   
   **host** - server host name. by default, host is 127.0.0.1
